CSS Files and Structure

This template is based of standards CSS3. The main template file style.css. It is located in the folder assets/css . With it, you can modify and customize the theme (such as fonts, colors, backgrounds, etc.). The file is separated into sections using: 

/***** General *****/
some code
/***** Typography & Element *****/
some code
/***** Header *****/
some code
/***** Social link*****/
some code
/***** Main Nav *****/
some codeetc, etc.

Template also uses additional files CSS (in the folder assets/css):

  • assets/css/bootstrap.css - Twitter Bootstrap framework
  • assets/css/font-awesome.css - Font Awesome
  • assets/css/prettyPhoto.css - jQuery prettyPhoto
  • assets/css/reset.css
  • etc.
This theme uses the free base fonts Google Web Fonts http://www.google.com/webfonts/. To use your fonts, you must edit the files with the .html extension line

 

JavaScript

This theme uses the jQuery framework and plugins: sliders, gallery, carousel, etc. These files are located in the folder assets/js. The main configuration file plugins and scripts based on jQuery is custom.js, it's in the folder assets/js. With it you can change the settings of the sliders, galleries, animations, etc.

Template also uses additional files js (in the folder assets/js):

  • assets/js/bootstrap.js - Twitter Bootstrap framework
  • assets/js/jquery.js - jQuery JavaScript Library
  • assets/js/jquery.easing.1.3.js - jQuery Easing plugin
  • assets/js/jquery.form.js - jQuery form plugin
  • assets/js/jquery.ufvalidator-1.0.5.js - jQuery validate form plugin
  • assets/js/jquery.prettyPhoto.js - jQuery prettyPhoto gallery plugin
  • assets/js/gmap3.min.js - jQuery Google Map plugin
  • assets/js/jquery.isotope.min.js - jQuery isotope plugin
  • assets/js/responsiveslides.min.js - jQuery responsiveslides plugin
  • etc.

For example, to change the address on Google Map, you need to open the file custom.js

and edit the line: 

//Google map top page setting

code:

$('#google_map_home').gMap({ address: 'Pearl St, NY', zoom:15,markers: [
					{ 'address' : 'Pearl St, NY',}
				] });

For example, to change slider setting, you need to open the file custom.js

and edit the line: 

// Slider plugin

code:

$(".rslides1").responsiveSlides({
        //auto: false,
        nav: true,
		pause: true,
        pager: true,
        speed: 2500,
		timeout: 5000,
        //maxwidth: 540
    });

For example, to change Media element player setting, you need to open the file custom.js

and edit the line: 

// Media element player

code:

$('audio,video').mediaelementplayer({
        audioWidth: '100%',
        audioHeight: '30px',
        videoWidth: '100%',
        videoHeight: '100%'
    });

PHP Files

To configure the sending of messages from the contact form on your site you need to edit the file contact.php in the folder /mail. Enter your e-mail address in the line:

$to = 'mail@yourmail.com'; //your e-mail

PSD Files

This theme is based on the standards of HTML5 and CSS3, so this theme is minimized using images for backgrounds, buttons, etc. Accordingly, the theme is not included the original PSD files as they are not needed.
